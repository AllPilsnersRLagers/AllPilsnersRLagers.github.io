Linux Tools I am Using¶

Terminal Tools¶

  • tldr - The tldr-pages project is a collection of community-maintained help pages for command-line tools, that aims to be a simpler, more approachable complement to traditional man pages. There is also a web based app called tldr InBrowser.App

  • duf - Disk Usage/Free Utility.

  • zoxide - is a smarter cd command for your terminal. It keeps track of the

     directories you use most frequently, and uses a ranking algorithm to
 navigate to the best match.

  • fzf - (Fuzzy Finder) is a lightning-fast, interactive command-line utility used to search and filter any list of items (files, command history, processes, git branches).

  • bat - cat with wings

  • gh - is GitHub on the command line. It brings pull requests, issues, and other GitHub concepts to the terminal next to where you are already working with git and your code.

  • htop - is an interactive, real-time process viewer and system monitor for Unix-like operating systems. It serves as a user-friendly, feature-rich alternative to the traditional top command, allowing you to easily manage system resources, monitor CPU/Memory usage, and kill or renice processes without needing their exact PIDs.

  • less - The less command in Linux is used to view the contents of a file one page at a time without opening it in an editor, making it ideal for reading large files efficiently.

  • pandoc - Pandoc is a Haskell library for converting from one markup format to another, and a command-line tool that uses this library.

  • ripgrep - is a line-oriented search tool that recursively searches the current directory for a regex pattern.

  • sed - GNU stream editor for filtering and transforming text.

  • tmux

  • EZA - s a modern alternative for the venerable file-listing command-line program ls that ships with Unix and Linux operating systems, giving it more features and better defaults. It uses colours to distinguish file types and metadata. It knows about symlinks, extended attributes, and Git.

  • calc - is an interactive calculator which provides for easy large numeric calculations, but which also can be easily programmed for difficult or long calculations. It can accept a command line argument, in which case it executes that single command and exits.

  • atuin = replaces your existing shell history with a SQLite database, and records additional context for your commands. Additionally, it provides optional and fully encrypted synchronisation of your history between machines, via an Atuin server.

  • yazi - is a terminal file manager written in Rust, based on non-blocking async I/O. It aims to provide an efficient, user-friendly, and customizable file management experience.

  • croc - is a tool that allows any two computers to simply and securely transfer files and folders.
